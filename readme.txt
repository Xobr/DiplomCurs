Програма дає можливість працювати у чотирьох режимах: 
 1. Режим нових слів(New Wrods)
 2. Режим слова із заданою частотою(F = ...)
   2.1 Як вхідни параметер програма приймає частоту слова(поле F = ) 
 3. Режим вибраного слова(Selected Word)
   3.1 Як вхідни параметер програма приймає значення слова(поле Word = )
 4. Режим вибраної Нграми(Selected NGram)
   4.1 Як вхідні параметри прорама приймає розмір нграми(Size) i її значення(Value) 
 5. Режим нграм(NGram)
   5.1 Як вхідні параметри програма приймає розмір Награми(Size) і тип Нграми(словесний, бкувений, символьний). Для буквеного і словесного доступна опція вибору стоп символів(Stop Symbols) 

У кожному із режимів присутні чотири влкдаки: 
 1. Analysis text - поувзцє лінгвістичний елемент, кілкість його появи в тексті і частота появи в тексті. 
 2. dTi - Показує графік і таблицю комулятивної і накопичувальної (p(x) i P(X)) функції для нових слів. 
 3. Ti - Комулятивна і накопичувальна функція для рівномірних проміжків тексту. Кількість проміжків задається із програми. 
 4. New Words - Комулятивна і накопичувальна функція для цілого тексту. 
 5. F.. - флуктуації(біжуче вікно). 
 
Особливості роботи програми: 
 1. Для збереження стоп символів після копіювання .exe файлу потрібно також скопіювати фали StopSC.txt i StopSL.txt у ту саму паку що і виконуваний файл. 
 2. У виавдку, якщо потрібно перегенрувати дані для якоїсь певної кладки ,для кращої швидкодії, краще перезагрузити програму. Без перезагрузки пргграма буде працювати коректно, але трошки довший період часу.
